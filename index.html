Customer Support AI Chatbot
RAG · NLP · Deployment

Identified a recurring bottleneck in a customer support team: agents were relying on memory and outdated documentation to answer questions, leading to slow, inconsistent responses. Built an end-to-end AI assistant — from raw audio to a live Telegram bot — entirely from scratch.

01
Speech-to-Text — Whisper
Transcribed years of customer service call recordings using OpenAI Whisper. Handled speaker diarization and filler word normalization at scale.
↓
02
Data Structuring & Chunking
Cleaned and tagged transcripts by issue type, resolution, and timeline. Chunked into semantically meaningful case summaries for retrieval.
↓
03
Vector Embeddings & Semantic Search
Generated embeddings from the structured corpus and indexed them in a vector database (FAISS / ChromaDB), enabling semantic retrieval over thousands of real past cases.
↓
04
RAG Architecture — LLM Layer
Built a Retrieval-Augmented Generation pipeline. Agent queries trigger context retrieval from the vector store; an LLM synthesizes grounded, accurate responses — no hallucinations, no static FAQs.
↓
05
Telegram Bot Deployment — Python
Deployed the full pipeline as a Telegram bot using Python (python-telegram-bot), giving support staff real-time access mid-conversation with zero new tooling required.
Reduced agent lookup time
Improved response consistency
Faster onboarding for new hires
Zero external budget
ATM Site Prospecting Tool
Google Places API · Mapping · CRM

Our business development process for finding new ATM host locations relied entirely on manual Google Maps searches — a platform that caps results at ~20 per view. The team had to zoom in, collect results, pan the map, and repeat across entire cities. It was slow, inconsistent, and guaranteed to miss locations. I built a web tool using AI that eliminated the manual search entirely, surfacing over 400 qualifying locations across key West Coast markets — replacing it with an automated grid-scanning system backed by a live map dashboard and built-in lead tracker.

01
Auto-Zoning & Grid Scan
The tool divides any target area into a systematic grid of zones and scans each one automatically via the Google Places API — bypassing the ~20 result cap that made manual search unscalable.
↓
02
Business Hours Filtering
Automatically filters results to only surface qualifying locations: open no later than 10am, close no earlier than 8pm, with 24-hour businesses flagged as priority targets — matching our ATM placement criteria exactly.
↓
03
Live Map with Location Pins
All qualifying businesses are rendered as color-coded pins on an interactive map, giving the team an instant visual overview of the entire prospect landscape across a city — something manual Google Maps searches could never provide.
↓
04
CRM Dashboard & Lead Tracking
A built-in dashboard tracks every location across three states: Potential (identified, not yet contacted), Contacted (outreach in progress), and Owned (ATM installed). Replaced scattered spreadsheets with a single source of truth for the BD team.
Eliminated manual map panning
400+ locations identified across West Coast
Built-in lead pipeline tracking
Powered by Google Places API
